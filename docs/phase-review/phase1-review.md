## ファイル構成

```
src/
├── main/           # Electron メインプロセス (5ファイル)
├── preload/        # プリロードスクリプト (2ファイル)
├── shared/         # 共有型定義 (1ファイル)
└── renderer/src/   # React フロントエンド
    ├── components/ # コンポーネント (3ファイル)
    ├── pages/      # ページ (7ファイル)
    ├── hooks/      # カスタムフック (1ファイル)
    └── utils/      # ユーティリティ (1ファイル)

data/
├── metadata/       # カテゴリ定義等 (2ファイル)
├── questions/      # 問題 JSON (5ファイル, 50問)
└── resume/         # レジュメ JSON (5ファイル)
```
